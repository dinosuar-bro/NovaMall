# 视图证据

## 目标

验证只读视图封装订单明细、有效销量和履约汇总，并与基础表查询结果一致。

## 视图清单

```text
v_effective_product_sales
v_member_order_details
v_shop_sales_summary
```

## 查询 SQL

```sql
SELECT *
FROM v_member_order_details
WHERE buyer_user_id = '155'
ORDER BY created_at DESC, shop_order_id
LIMIT 5;

SELECT COUNT(*) AS view_rows,
       SUM(CAST(line_amount AS DECIMAL(12,2))) AS view_amount
FROM v_member_order_details
WHERE buyer_user_id = '155';

SELECT COUNT(*) AS base_rows,
       SUM(oi.line_amount) AS base_amount
FROM master_orders mo
JOIN shop_orders so ON so.master_order_id = mo.id
JOIN order_items oi ON oi.shop_order_id = so.id
WHERE mo.buyer_user_id = 155;

SELECT *
FROM v_shop_sales_summary
ORDER BY sales_date DESC, shop_id
LIMIT 5;
```
